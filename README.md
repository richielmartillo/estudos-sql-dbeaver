# Estudos SQL com DBeaver

Este repositório contém consultas SQL praticadas no DBeaver utilizando o banco de dados de exemplo SQLite / Chinook.

## Objetivo

Praticar comandos básicos e intermediários de SQL, incluindo:

- SELECT
- WHERE
- ORDER BY
- INNER JOIN
- GROUP BY
- COUNT
- LIKE

## Arquivo da entrega

O arquivo principal da entrega da atividade é:

`00-Atividade-Entrega.sql`

Esse arquivo contém as consultas SQL solicitadas no exercício da EBAC.

## Arquivos de estudo

Os arquivos abaixo foram utilizados para prática durante as aulas:

01-Aula-SQL-Script.sql
02-Aula-SQL-Script.sql
03-Aula-SQL-Script.sql

## Banco de dados utilizado

Foi utilizado o banco de dados de exemplo do DBeaver:

`DBeaver Sample Database (SQLite)`

Também conhecido como banco Chinook.

## Consultas realizadas na entrega

A atividade contém consultas para:

1. Selecionar artistas em ordem alfabética.
2. Listar músicas com duração maior que 5 minutos.
3. Selecionar músicas do gênero Rock.
4. Listar álbuns junto com o artista correspondente.
5. Selecionar músicas do álbum "Let There Be Rock".
6. Listar clientes que moram no Brasil.
7. Exibir o total de músicas por gênero, do maior para o menor.
8. Selecionar músicas que possuem a palavra "Love" no nome.
